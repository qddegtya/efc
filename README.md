### About

> 🔧 Elegant devtool for formily component.

### Requirements

* node version >= 8.10

### Features

* 🚀 init / dev / build / publish 一键式体验
* ⚙️ 自动安装依赖
* 🌍 完美支持 HMR

### Install

```bash
> npm install @atools/efc -g
```

### Commands

#### init

```bash
> efc init
```

#### dev

```bash
> efc dev
```
